FAQ

자주 묻는 질문

도입을 검토하시는 분들이 가장 많이 물어보시는 질문들을 모았습니다.

도입 AI 도입에 필요한 장비나 하드웨어를 별도로 구입해야 하나요?
별도 하드웨어 구입이 필요하지 않습니다. AX덴탈그룹의 AI 솔루션은 클라우드 기반으로 제공되며, 기존 PC나 태블릿에서 웹 브라우저를 통해 바로 사용하실 수 있습니다. 다만, X-ray 분석 AI의 경우 기존 디지털 X-ray 장비(RVG, CBCT 등)와의 연동이 필요하며, 이는 설치 과정에서 AX덴탈 기술팀이 지원합니다.
보안 환자 데이터는 안전하게 보호되나요?
환자 데이터 보안은 최우선 과제입니다. AX덴탈그룹은 의료정보보호법(개인정보보호법, 의료법)을 완전히 준수하며, 다음과 같은 보안 체계를 갖추고 있습니다:

• AES-256 암호화로 모든 데이터 저장
• TLS 1.3 프로토콜로 전송 중 데이터 보호
• 국내 의료 데이터 센터 전용 서버 운영 (해외 반출 없음)
• ISO 27001 정보보안 인증 획득
• 접근 권한 관리: 직급별 차등 접근 제어
교육 직원 교육은 얼마나 걸리나요? 어렵지 않나요?
교육은 평균 2~3일(16~24시간) 과정으로 진행됩니다. 치과 스태프가 IT 전문 지식 없이도 쉽게 익힐 수 있도록 직관적인 UI로 설계되었습니다. 교육은 온라인 동영상 자료 + 오프라인 현장 교육 + 이후 1개월 집중 지원으로 구성됩니다. 교육 후에도 24시간 챗봇 지원 및 전담 매니저를 통해 언제든 도움을 받을 수 있습니다.
비용 도입 비용은 어떻게 되나요?
도입 비용은 치과 규모와 선택하는 패키지에 따라 달라집니다. 초기 설치비 + 월정액 구독 방식으로 운영됩니다. 도입 문의를 통해 무료 컨설팅을 신청하시면 치과에 최적화된 맞춤 견적을 받아보실 수 있습니다. 평균 ROI는 도입 후 6개월 이내 손익분기점을 돌파하며, 1년 내 투자 회수가 가능한 수준입니다.
기능 기존에 사용하던 치과 프로그램(EMR/OCS)을 교체해야 하나요?
교체할 필요가 없습니다. AX덴탈그룹 AI 플랫폼은 기존 EMR/OCS 시스템 위에 AI 레이어를 추가하는 방식으로 동작합니다. 덴탈포스, DentOCS, SmileDent 등 주요 치과 소프트웨어와 API 연동을 통해 데이터를 실시간 동기화합니다. 기존 데이터와 워크플로우는 그대로 유지하면서 AI 기능만 추가되는 방식입니다.
성과 실제로 노쇼가 줄어드나요? 어떻게 예측하나요?
실제 도입 치과에서 평균 노쇼율 40% 감소를 달성했습니다. AI 노쇼 예측 엔진은 환자의 예약 이력, 취소 패턴, 날씨·요일·시간대 데이터를 결합하여 노쇼 가능성을 예측합니다. 노쇼 고위험 환자에게는 예약 전날 자동 리마인더 문자를 발송하고, 당일 오전 재확인 알림을 보냅니다. 또한 노쇼 발생 시 대기 환자를 자동으로 연락하여 예약 빈자리를 채웁니다.
글로벌 해외 진출을 계획 중인데 해외에서도 사용 가능한가요?
네, AX덴탈그룹은 글로벌 확장을 핵심 전략으로 추진하고 있습니다. 현재 동남아(베트남, 태국, 필리핀), 중동(UAE, 사우디아라비아), 미주 지역을 대상으로 파트너십을 구축 중입니다. 다국어 지원(한국어, 영어, 베트남어, 아랍어), 각국 의료 규제 준수, 현지 EMR 연동을 지원합니다. 해외 진출 계획이 있으신 경우 엔터프라이즈 패키지를 통해 글로벌 확장 지원을 받으실 수 있습니다.
